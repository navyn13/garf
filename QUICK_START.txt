GARFIELD COIN - QUICK START GUIDE
==================================

🚀 GET STARTED IN 3 STEPS
--------------------------

STEP 1: Setup MongoDB
---------------------
Choose one option:

A) MongoDB Atlas (Recommended - Free & Easy):
   1. Go to: https://www.mongodb.com/cloud/atlas
   2. Sign up for free account
   3. Create cluster (M0 Free tier)
   4. Click "Connect" → "Connect your application"
   5. Copy connection string
   6. Create .env.local file with:
      MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/garfield_game

B) Local MongoDB:
   1. Install: brew install mongodb-community
   2. Start: brew services start mongodb-community
   3. Create .env.local file with:
      MONGODB_URI=mongodb://localhost:27017/garfield_game

STEP 2: Run Development Server
-------------------------------
npm run dev

STEP 3: Open Browser
---------------------
http://localhost:3000

That's it! 🎉

WHAT YOU'LL SEE
---------------
✓ Landing page at http://localhost:3000
✓ Game page at http://localhost:3000/game
✓ Connect Phantom wallet to play
✓ Collect dollars, avoid obstacles
✓ Compete on leaderboard
✓ Top 50 can claim rewards

TESTING THE GAME
----------------
1. Open http://localhost:3000
2. Click "PLAY GAME" button
3. Install Phantom wallet if needed: https://phantom.app
4. Click "Connect Wallet"
5. Press SPACE or TAP to jump
6. Collect green circles (💵)
7. Avoid red rectangles
8. Game over → score submitted automatically

VERIFY IT'S WORKING
-------------------
✓ Landing page loads
✓ Countdown timer is running
✓ Stats show "0 Players Today, 0 Top Score"
✓ Game page loads
✓ Wallet connect button appears
✓ After connecting: game canvas appears
✓ After game over: leaderboard updates

COMMON ISSUES
-------------
Q: "Cannot connect to MongoDB"
A: Check your MONGODB_URI in .env.local

Q: "Wallet not connecting"
A: Install Phantom wallet extension

Q: "Game not loading"
A: Check browser console for errors

Q: "Leaderboard empty"
A: Play a game first to add scores
